```
screen对象包含有关客户端显示屏幕的信息
```

```
screen对象属性

语法：screen.availWidth

功能：返回可用的屏幕宽度

语法：screen.availHeight

功能：返回可用的屏幕高度

获取当前浏览器打开的窗口文档显示区的高度和宽度，可以使用window.innerHeight和window.innerWidth
获取显示屏幕的高度和宽度，可以使用screen.availWidth和screen.availHeight
```
