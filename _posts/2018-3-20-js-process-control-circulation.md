```
for语句

语法：
for(语句1;语句2;语句3){
    被执行的代码块
}

语句1：在循环（代码块）开始前执行
语法2：定义运行循环（代码块）的条件
语法3：再循环（代码块）已被执行之后执行
```

```
while语句

语法：
声明初始值
while(条件){
    需要执行的代码；
    更改执行条件
}
```

```
for与while的区别

for 适合已知循环次数的循环体
while 适合未知循环次数的循环体
```
