### 跳槽经历

```
之前换工作都是为什么？

虽然实话实说好，比如说公司文化不好老让加班呆着不爽，这么说出来HR这轮估计就要挂掉了，说的很实在，别人都能理解，但是HR是站在公司利益审核方，
他不能通过。所以跳槽的经历这块特别忌讳上一家公司不好，是大忌

以上客观事实不能说出来，那该怎么去回答这个问题呢，技巧又是什么？

应该告诉HR换工作就是因为自己有一个明确的职业规划，比如说自己在几年之内要达到什么样的成就，这种成就可以是技术上的也可以是管理上的也可以是工作和生活相平衡，总之
就是要说之前的工作尤其是某一份工作和自己的个人规划不太相符，经过各种思索和对现实情况的考虑，最后做出的抉择是选择进行下一份公司的一个挑战

上面这种方法说出来HR就会舒服很多，为什么会感觉舒服呢？
第一：HR不会感觉你是一个自私的人（太累会离职、不爽会离职那上公司公司敢把重要任务交给面试者吗，工作都会累，累了就会离职，所以才会让人觉得自私，
没有承担责任的勇气，刚才那么回答既不会让HR觉得自私，也会让HR认为你是一个有明确职业规划的人）
```

### 频繁的跳槽

```
在整个HR的世界里，频繁跳槽的人是靠不住的，类似于谈感情问题一样，公司招聘一个人的成本是很高的。如果不可避免的在你的过去职业中真的就是这样频繁的跳槽那怎么办？
话术技巧简历这块可以去装饰一下，还有一方面就是自己一定要思考自己的一些问题：

1、去一个公司三个月内发现这个公司很多的问题然后觉得自己不得已必须离职，但是自己一定要反思在面试之前没有看清楚这个公司，这个是自身的问题，简历不要乱投，因为每一次机会对个人职业生涯都有严重的影响，
频繁的跳槽就是因为自己之前没有认真谨慎去分析那些公司，结果进公司之后发现各种不如意，所以自己这方面的成长也是必须做到的。
2、在简历过程中尽可能合并这些跳槽，最好的是在这些公司中至少呆满一年，这是一个规则
```

### 薪资相关

#### 之前的待遇问题

```
1、尽量去打听工资薪酬范围，这个一般情况下是保密的，可以去打听关系好的人，但是关系好的还不一定会告诉
```

#### 你对当前待遇的期望

```
1、要低了吃亏，要高了怕人家不要
2、像阿里、腾讯、去哪大公司什么样的级别对应着什么样的薪资范畴
3、争取把之前的待遇说的高点，但是这个高也不能高的太离谱，聊的过程中尽量自己不要去说这个数，可以向如下去说：

咱们公司反正也是非常知名的公司，做事情也很规范我也清楚，然后在薪资这块呢，通过我几轮的技术面试，和您和我聊得各方面的一个
面试考察，我觉得呢对我是什么样的安排，我觉得咱们公司能做出比较合理的决定，然后我还是愿意遵守公司的规则，然后我更看重这个
平台和机会

一般按照以上方式去说一般的HR不会强逼着你去说一个数了，他会觉得你非常的机制，如果你说出具体的一个数来，说实话要的低了，已经
没有办法要的高了，HR就顺坡下了，比如你之前15你说我要16吧，其实公司能给到20，那你要的16他给你16，offer给你发了，到了公司你一看别人都20，我怎么16啊，
说实话你已经无法改变了，但是如果像刚才那么说的话，HR会通过你之前的技术面试，一商议，那么他可能会给你一个他们公司的均价，所以尽量不要去说这个数，
但是如果把刚才说的话说了一遍HR还是要求你说一个数，那我觉得按照你当前的薪资还有你期望的薪资，中间打一个89折或85折
```

#### 如果达不到这个期望，你会如何选择

```
如果说这个工资达不到我就要考虑考虑了，HR可能就会放弃你，他会觉得你太看重钱，而不看中他们公司的业务和技术氛围，这个是HR不太喜欢的点，因为有的CEO特反感
上来就谈钱的候选人，他就显的你没有情怀，虽然大家都知道没有钱光谈情怀有什么用，但是公众场合大家还是要注意你只提钱，尤其是这个岗位竞争者很多的时候，一些聪明的
候选人他也喜欢钱，但是他描述的可能就比你委婉，就胜出了，遇到这样的问题，还是就说不要谈一个具体的数，还是要说咱们公司是规范的是有标准的，只要按照这个标准给我
就行，那么一般情况下给你的就不会太低，别人20他基本也会给你20，你要优秀的话他可能给你22，这个东西没有具体的数，只是个参考，横向参考，如果非要让你说达不到期望你会
怎么办，那你积极乐观一点，你说，我相信公司不会让我失望的，还有一种就是如果达不到我的期望，那么说明我的技术还不到咱们公司的一个要求，我会更加努力达到目标，
你要是这么说，HR就会觉得你非常的积极向上正能量，当然了他不会因为你达不到技术水平不够就压你的薪资，HR不会这样做的，聊完之后薪资这块基本已经定了。

薪资这块值得认真去思考一下，不要太冲动，也不要表现的太直接
```


#### 4、各种杳无音信，积极性受挫

```
自身找问题，不要有情绪化，任何情绪都没什么用，要找到自身的问题，找到问题并解决，这才是正确的方式方法，一旦有这种方式方法你的情绪就不会那么容易被影响到
```

#### 5、面试被稳住，消极对待

```
面试过程中被稳住太正常了，其一是面试官的工作经验可能比你长，其二他所问的问题都是他所擅长的，也就意味着不是你擅长的，那么他问住你是不是很正常
其三面试官问你的问题不决定你将来面试的结果，如果说面试官问了你一个问题但是你没有回答上来，记住专业的面试官问你问题他不纠结于问题的答案，而是取决于
回答这个问题中间思考的过程，如果说面试官问了你一个问题你一想不会然后立马说不会，这个就是不对的，如果说你会然后你就说你会那这还不是一个优秀的
情况，面试官尤其是专业的面试官会特别注意你在拿了这个问题以后你有没有思考你的思维方式是什么，你的态度是什么，这些远比问题本身答案更有价值
所以面试中被问住了一点也不可怕，这个真的很正常，真不取决于某几个问题被问住，那么offer就不给你了，这个真不一定，不要担心，遇到这种情况一定要冷静，
你就说面试官能不能给我一点时间让我想一下、思考一下，这个是一个非常正向的结果，最后如果思考了半天但真的没有思路，这个没关系，只要你思考了这个过程就够了
或者是说思考过程中我这种思路是不是对的，至少你让面试官知道是可以思考问题是有方向的是有思维方式的
```


#### 6、感觉聊得差不多，最后还是没通过

```
多数情况下是你自我感觉良好，有些表现行为你自己没注意到问题，比如说HR问你为什么会离职啊，然后你说我工作很不爽然后HR会对着你笑一笑，这时你可能会觉得我聊的不错啊，
他都跟我笑了，你觉得这个笑背后有几层意思，有时候你觉得聊得差不多是你自己的感受，HR是一些专业的面试官，在面试过程中不会当场给你一些反馈，不会任何反驳你，
面试官和HR会问你很多问题，他们最后是听，听完之后不会给任何反馈，有时面试太顺利反而不是什么好事，所以说聊得差不多但没反馈然后感觉很郁闷很纠结遇到这种情况完全没必要

```
